# Aplicação WEB em Cloud
Neste repositório, apresento uma aplicação WEB em nuvem utilizando a AWS. Foram utilizadas uma instância EC2 para hospedar o servidor, e uma RDS para o banco de dados MySql.

# Estrutura

│   package.json
│
